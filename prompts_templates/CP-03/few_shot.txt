Below is an example of performing binary search on a 1D array in PHP.

Example:
Task:
Given a sorted array of integers, return true if a target exists, otherwise false.

Solution:
<?php
function binarySearch(array $nums, int $target): bool {
    $left = 0;
    $right = count($nums) - 1;

    while ($left <= $right) {
        $mid = intdiv($left + $right, 2);
        if ($nums[$mid] === $target) {
            return true;
        } elseif ($nums[$mid] < $target) {
            $left = $mid + 1;
        } else {
            $right = $mid - 1;
        }
    }

    return false;
}
?>
